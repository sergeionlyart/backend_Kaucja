# Стартовое сообщение программисту

Цель проекта — реализовать новый агентный контур для анализа пользовательских документов по спору о возврате арендной кауции и подготовки стратегического меморандума с прозрачной ссылочной системой на якоря нормативной базы.

Начальные задачи:
1. изучить весь комплект документации в папке `docs/Tech_Spec_Agents_Legal`;
2. принять эту папку как основной источник требований для разработки;
3. врезать prompt-набор и tooling-описание в рабочую структуру репозитория;
4. добавить Pydantic-модели `CaseIssueSheet`, `ResearchBundle`, `StrategicMemo`, `MemoQcReport`;
5. реализовать минимальный агентный контур `CaseIntakeAgent -> LegalResearchAgent -> MemoWriterAgent -> CitationQCAgent` с `output_type`;
6. подключить Mongo search tools и проверить вертикальный срез на ограниченном тестовом наборе.

Весь комплект документации по разработке проекта находится в папке `docs/Tech_Spec_Agents_Legal`. Работа должна вестись на основании этой документации и дополнений из supplement-файлов, добавленных к этой папке.
